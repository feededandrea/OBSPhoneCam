# Instagram Automation

## Enfoque realista

La app no debe prometer live streaming directo a Instagram si no existe un flujo oficial disponible para la cuenta/permisos.

## Modos

1. Manual Export
   - Preparar archivo.
   - Copiar caption.
   - Revelar en Finder/Share Sheet.

2. Content Publishing API
   - Publicar clips/Reels con OAuth y permisos oficiales.
   - Requiere configuración Meta Developer y App Review.

3. Live Experimental
   - Solo si existe API/permiso oficial validado.
   - Por defecto no habilitado.

## Flujo recomendado

OBS Replay Buffer -> Clip Library -> Preparar Instagram -> Publicar si API habilitada o exportar manual.
