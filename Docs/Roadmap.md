# Roadmap

## Etapa 1 - MVP OBS Remote

- Mac Hub conectado a OBS.
- iOS envía comandos al Mac Hub.
- Cambiar escenas.
- Guardar Replay Buffer.

## Etapa 2 - Video iPhone a Mac

- Captura AVFoundation.
- Encoder H.264.
- Transporte local.
- Preview Mac.

## Etapa 3 - Reconexión robusta y múltiples dispositivos

- DeviceSession final.
- Bonjour listener.
- Heartbeats.
- Métricas reales.

## Etapa 4 - Cámara virtual

- CoreMediaIO Camera Extension.
- SampleBufferProvider.
- Varias cámaras virtuales.

## Etapa 5 - Clips

- SaveReplayBuffer.
- Metadata.
- ClipTrimmerService.

## Etapa 6 - Instagram

- Manual export.
- OAuth.
- Content Publishing API si permisos correctos.

## Etapa 7 - Optimización

- Adaptive bitrate.
- Audio sync.
- Mejor UI.
- Monitoreo de performance.
